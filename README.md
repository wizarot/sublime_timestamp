Sublime 2/3时间戳插件
=========

更方便的插入时间戳


## Install

- `菜单` `Tools` -> `New Plugin...`,
- 粘贴 `timeStamp.py` 的文本内容, 保存到扩展的安装目录即可使用


## Usage

key + <kbd>tab</kbd>

| *key*  | *format* | *example* |
| ------  | -------- |  -------- |
| `now` | `Y-m-d H:i:s` | `2014-11-22 14:41:54` | 
| `tsnow` | timestamp | `140xxxxxxx` |
| `2014-12-22-05-5-23` | timestamp | `140xxxxxxx` |
